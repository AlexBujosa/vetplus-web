# How to install

1. Install the packages

```bash
$ npm install
```

2. Open the project

```bash
npm run dev
```

3. Go to the login page http://localhost:5173/login [Login Page](http://localhost:5173/login)

## Run DEV environment

```bash
$ docker compose -f docker-compose-dev.yml up -d
```

## Run QA environment

```bash
$ docker compose -f docker-compose-qa.yml up -d
```
